                          Tiny Tile Editor

  What is it?
  -----------

  Written in C# 4.0 and utilizing the XNA Framework 4.0, Tiny Tile 
  Editor allows its users to quickly and easily create simple and 
  adaptable two-dimensional tile maps for games. Supporting multipurpose 
  drawing tools, multiple layers, a robust undo-redo engine, the capability 
  to dynamically change the map's current dimensions, and a streamlined 
  user interface and design, Tiny Tile Editor is both tiny and powerful.

  Requirements
  -----------

     o Microsoft Windows Operating System

     o Microsoft .NET Framework Redistributable 4.0

     o Microsoft XNA Framework Redistributable 4.0

  The Latest Version
  ------------------

  Details of the latest version can be found on the Tiny Tile Editor
  project page under http://code.google.com/p/tiny-tile-editor/.

  Installation
  ------------

  This application is standalone. Simply run the executable.
